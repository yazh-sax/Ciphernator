• Ciphers








The date shift is a cipher that employs a different technique than many other ciphers. Instead of using a traditional key, the date shift cipher uses a significant date. This can be useful because transmission of a key to another person as long as a memorable date is agreed upon. In that respect, the date shift cipher offers more protection than other simple encryption methods.

This tool works by taking in a date and converting it to a shiftable sequence of numbers. This sequence changes when the date changes. Then, the message is translated using that sequence of numbers. Finally, the encrypted message is displayed. To decrypt, fill the message box with the encrypted message, and the date box with the date used to encrypt it.
